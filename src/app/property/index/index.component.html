About this website

This is a website that I developed in order to improve as a developer. It is my first website developed with angular and I had really strict time limit, but I managed to go over some of the key aspects of angular development. Key aspects of this website are:

  • Angular routing
  • Event emitters
  • Input
  • Services
  • Api calls
  • Observables
  • Style binding
  • External libraries
  • Pipes

There are some advanced topics in angular that I havent covered yet. I am already avare of these things and I will soon create a project of some of the more advanced angular features. These features include:

  • rxjs
  • Angular animations
  • Dynamic routing

This was not designed to be visual website but I still used some sass and bootstrap to get accustomed to them.
